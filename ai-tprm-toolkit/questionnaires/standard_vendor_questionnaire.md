## Section 2 — Cybersecurity Controls

### 2.1 Access Control
- [ ] Principle of least privilege enforced for all accounts?
- [ ] MFA enforced for all remote access?
- [ ] MFA enforced for all privileged accounts?
- [ ] Documented joiners/movers/leavers (JML) process?
- [ ] Accounts disabled on employee departure within same day?

### 2.2 Vulnerability Management
- [ ] Documented vulnerability management programme?
- [ ] Vulnerability scanning frequency? (Minimum: monthly)
- [ ] Patching SLA for critical vulnerabilities? (Target: <72 hours)
- [ ] Last external penetration test? (Minimum: annually for Critical tier)

### 2.3 Logging and Monitoring
- [ ] Security event logs maintained for all systems processing our data?
- [ ] Log retention period? (Minimum: 12 months)
- [ ] SOC or equivalent monitoring capability?
- [ ] SIEM deployed?

### 2.4 Endpoint and Network
- [ ] EDR deployed on all endpoints?
- [ ] Network segmentation between security zones?
- [ ] DLP controls in place?
- [ ] Email security filtering deployed?
