## Step 2 — High-Risk System Obligations (If Annex III)

### Risk Management System (Article 9)
- [ ] Risk management system established for this AI system?
- [ ] Risks identified, estimated, and evaluated?
- [ ] Risk management measures implemented and tested?
- [ ] System tested against reasonably foreseeable misuse?

### Data Governance (Article 10)
- [ ] Training, validation, and testing datasets documented?
- [ ] Data governance practices documented?
- [ ] Datasets examined for biases?
- [ ] Data relevant, representative, and accurate?

### Technical Documentation (Article 11)
- [ ] Technical file maintained and up to date?
- [ ] Available for competent authority inspection on request?

### Transparency and Instructions for Use (Article 13)
- [ ] System designed to be sufficiently transparent?
- [ ] Instructions for use provided to deployer?
- [ ] Instructions cover intended purpose, performance, limitations?

### Human Oversight (Article 14)
- [ ] System designed for effective human oversight?
- [ ] Humans can monitor, override, and stop the system?
- [ ] Risks of automation bias addressed?

### Accuracy, Robustness, Cybersecurity (Article 15)
- [ ] Accuracy levels documented and tested?
- [ ] System robust against errors, faults, and adversarial inputs?
- [ ] Cybersecurity measures documented?
- [ ] Performance consistency across diverse groups tested?

### Conformity Assessment (Article 43)
- [ ] Conformity assessment conducted?
- [ ] CE marking applied (where applicable)?
- [ ] EU Declaration of Conformity available?
- [ ] System registered in EU database (Article 71)?

---

## Step 3 — GPAI Model Obligations (If applicable)

- [ ] Technical documentation prepared (Article 53)?
- [ ] Copyright compliance policy in place?
- [ ] Summary of training data published?
- [ ] For systemic risk models: adversarial testing conducted (Article 55)?

---

## Step 4 — Transparency Obligations (Limited Risk)

- [ ] Users informed they are interacting with an AI system?
- [ ] AI-generated content labelled (deepfakes, synthetic content)?

---

## Step 5 — General Obligations (All AI systems)

- [ ] AI literacy ensured for staff operating the system (Article 4)?
- [ ] Post-market monitoring in place?
- [ ] Serious incident reporting procedure established?
